## Horse Racing — Vue 3 + TypeScript + Vuex

Demo game “Horses Racing”.

### Install

```bash
npm i
# if vuex doesn't install automatically:
npm i vuex@^4
```

### Runing scripts

```bash
npm run dev
npm run build
npm run preview
```

### Short description

- Dynamically generated list of horses. 20 horses with unique colors and condition score(1–100).
- The 'Generate Program' button creates six races with distances of 1200–2200 meters and 10 random participants in each.
- 'Start' starts the races one by one; the results of each race appear in the 'Results' section upon completion.
- 'On the track,' the horses animate as they race.
